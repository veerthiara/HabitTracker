# Phase 11 — Feedback and Learning Loop

## Goal
Capture user feedback on answers and use it to improve prompts, routing, and answer quality.

## Why
The system currently answers questions but does not learn from incorrect or unhelpful responses.

## Scope
- feedback schema and API
- frontend thumbs up/down UI
- trace-to-feedback linkage
- offline analysis of poor responses
- iterative tuning workflow

## Out of scope
- online training/fine-tuning
- automated model retraining
